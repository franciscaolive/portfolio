Dress Up Game

Pick your style!

This project was developed as part of my Interface Design course subject, which proposed creating an experimental web project that explores different navigation modes, the concept of hypermedia, non-linear narrative, and applies HTML and JavaScript. Essentially, it aims to develop something that goes beyond the common strategies used by conventional web pages that have a predominantly informative function.

The experience begins outside a shopping mall. On the next page, we have the interior of said mall, which has 4 stores. By clicking on each one, we arrive at each store's page. In each store, the user needs to find the number of items indicated by the counter at the top of the page. To find them, the player needs to search with the mouse for red tags. At the end of the fourth store, we see a button that takes us to the page where the player can choose the clothing style they most identify with and then dress the character.

This project was developed in collaboration with my colleague Alice Logvina, who handled the first part in the mall, producing the visual components and coding the first half.

I worked on developing the second part of the game, drawing the visual components in Procreate and coded that entire section.
